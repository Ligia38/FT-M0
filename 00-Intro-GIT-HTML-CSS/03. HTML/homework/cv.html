EXPERIENCIA

Julio 2018 hasta Diciembre 2019

Trabajo de invertigación en la Universidad Central de Venezuela

Diseño de un prototipo para la determinación de la velocidad de onda de pulso en el sistema cardivascular

El principal objetivo de esta investigación fue el desarrollo de un prototipo para la obtención simultánea de las señales de fotopletismografía (PPG) y electrocardiografía (ECG), con el propósito de determinar el tiempo de propagación de la onda de pulso (PTT) en el sistema cardiovascular y su correspondiente velocidad promedio. La morfología de estas dos señales normalmente se utiliza para caracterizar al sistema cardíaco y vascular por separado; Sin embargo, estos dos sistemas se encuentran acoplados. ESte dispositivo, además de proporcionar los registros para la posterior investigación sobre el acople entre estos dos sub-sistemas, puede ser utilizado para caracterizar el funcionamiento del sistema en general.
